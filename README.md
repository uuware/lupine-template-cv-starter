# lupine-template-cv-starter

A CV Starter to create your own CV. You can also use `npx create-lupine@latest my-cv` and select the **`cv-starter`** template when prompted.

## Usage

This repository serves as a template for the documentation framework. To create a new documentation project, please use:

```bash
npx create-lupine@latest my-docs
# Select 'doc-starter' template
```

For detailed instructions, please refer to:
https://uuware.github.io/lupine.js/en/articles/introduce-lupine.press
